Context-Optimized Research Agents

ü¶Å Gryffindor Agent: The Advocate

Context Engineering Strategy:

  • Priority Context: Real-world impact metrics, case studies, policy documents
  • Prompt Engineering: "Identify actionable interventions that..."
  • Memory Management: Maintains history of successful advocacy strategies
  • Tool Integration: Community feedback APIs, policy tracking systems
context_window = prioritize(community_voices, impact_data)

ü¶° Hufflepuff Agent: The Collaborator

Context Engineering Strategy:

  • Priority Context: Stakeholder perspectives, consensus protocols, ethical guidelines
  • Prompt Engineering: "Consider all viewpoints and find common ground..."
  • Memory Management: Tracks participation patterns and inclusion metrics
  • Tool Integration: Collaboration platforms, sentiment analysis
context_window = balance(stakeholder_inputs, fairness_metrics)

ü¶Ö Ravenclaw Agent: The Analyst

Context Engineering Strategy:

  • Priority Context: Academic papers, datasets, methodological frameworks
  • Prompt Engineering: "Analyze patterns and derive theoretical insights..."
  • Memory Management: Builds knowledge graphs and citation networks
  • Tool Integration: Statistical packages, literature databases
context_window = structure(academic_sources, theoretical_models)

üêç Slytherin Agent: The Strategist

Context Engineering Strategy:

  • Priority Context: Power structures, decision-maker profiles, leverage points
  • Prompt Engineering: "Identify strategic opportunities to maximize..."
  • Memory Management: Maps influence networks and tracks outcomes
  • Tool Integration: Network analysis, game theory models
context_window = optimize(influence_maps, outcome_probabilities)
